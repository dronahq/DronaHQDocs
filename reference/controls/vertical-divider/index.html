Vertical Divider

The Vertical Divider control is used to visually separate sections within a microapp by displaying a vertical line, creating a visual distinction between different content areas.

Properties

PropertyDescription
WidthSet the width of the divider.
Border TypeSelect from a range of border types such as double, dashed, solid and more.
Font ColorSet the color of the vertical border.
